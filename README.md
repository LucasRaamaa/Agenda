# 📇 Agenda de Contactos (Java)

---

##  Objetivo del proyecto

Practicar el uso de:
- Clases, objetos y encapsulamiento.
- Arreglos dinámicos mediante `ArrayList`.
- Estructuras de control (`while`, `switch`, `if`).
- Métodos organizados por responsabilidad.
- Entrada y salida de datos por consola.
- Separación en capas (modelo, servicio, vista).

---

##  Funcionalidades principales
- Agregar contacto  
- Listar contactos  
- Buscar contacto por nombre  
- Modificar datos de un contacto  
- Eliminar contacto
